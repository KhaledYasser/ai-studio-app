# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/1de90d58-efbf-4703-aa83-203e671630dd

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## Games Collection

The AI Studio App now features a fully integrated games collection:

- **Snake**: The classic grow-as-you-eat game.
- **Invadors**: Retro space shooter.
- **Tetris**: Block stacking puzzle.
- **2048**: Number merging challenge.
- **Flappy Bird**: One-tap flight simulator.
- **Slimy Soccer**: Physics-based sports game against AI.

All games are built with React and Canvas API, optimized for both desktop and mobile.
